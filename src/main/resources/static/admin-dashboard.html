OPERATIONS

教務、學生、家長與薪資集中管理。

主任主控台用來掌握每日課表、班級點名、學生分班、家長聯絡與老師薪資摘要。老師端則獨立成工作台，避免現場操作時進到主任專用資料。

報名諮詢 課表匯入 學生分班 LINE 整合 薪資管理 系統檢查

營運模組狀態

課表與教材進度Excel 匯入後可依日期、班級、老師與教材查詢。
班級點名電腦與手機皆可儲存出缺席、遲到時間與備註。
家長聯絡資料電話、LINE ID、接送備註與緊急聯絡集中整理。
LINE 整合中心已建立狀態中心；正式推播待 Channel Token、Secret 與 LIFF 設定。
薪資管理可登錄老師工時、時薪與月份薪資摘要。

報名諮詢

前台線上報名、試聽詢問與新生需求集中管理，可標記已聯絡、需追蹤與已入班。

數位互動課表

匯入 Daily Syllabus，依日期查看課程、老師、教材與核准狀態。

學習內容

發布今日單字、句型與作業提醒，讓學生與家長能在前台學習中心複習。

學生資料中心

管理學生名單、班級分發、學校年級與家長 LINE 綁定狀態。

班級點名

儲存準時、遲到、請假、曠課與老師備註，支援日期回查。

請假審核

櫃台代登假單、主任核准退回，核准後同步到班級點名。

家長聯絡資料

整理電話、LINE ID、接送備註與緊急聯絡資料。

LINE 整合中心

查看家長 LINE 覆蓋率、推播設定、通知佇列與訊息模板草稿。

薪資管理

依老師、日期、時數與時薪彙整薪資，後續可接課表自動帶入。

老師工作台

預覽老師每天使用的課表、電腦點名與手機點名入口。

系統健康檢查

集中查看資料庫、LINE、課表批次、學生分班與 NAS 上線前狀態。

營運摘要

學生與分班學生、班級與分班人次由課表/名單匯入後同步。
--
家長 LINE 資料缺 LINE ID 的學生會集中在家長聯絡資料中處理。
--
LINE 推播模式沒有 Channel Token 前，點名只會儲存草稿與狀態，不會發送。
--
老師薪資本月登錄時數與薪資估算會進入薪資管理彙整。
--

行政流程

課表匯入主任或老師匯入 Excel，建立每日課程與班級資料。
學生分班Reading Assignment 名單形成學生與班級關聯。
點名與聯絡老師點名後，主任可接續處理家長聯絡與請假。
薪資彙整老師工時、時薪與課程備註集中產生月份薪資摘要。
